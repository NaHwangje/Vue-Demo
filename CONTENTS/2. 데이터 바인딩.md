# 데이터 바인딩
  - {{데이터바인딩}}
    - 
    - 데이터를 HTML에 꽂아넣는 문법
    - 데이터 보관함부터 있어야함. (데이터 보관함에서 꺼내옴)
        ```    
        <script>
            export default {
            name: "App",
            data() {
                return {
                price1 : 60,
                };
            },
            components: {},
            };
        </script>
        ```
    - HTML 태그에 포함시키면 보관함에서 알아서 가져와서 출력해줌
        ```html
        <div>
            <h4 >{{ price1 }}</h4>
            <p > 100 만원</p>
        </div>
        ```
